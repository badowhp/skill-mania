# Deliberate Design Process

Use this when no clear brand or reference exists. The goal is not to find a universally tasteful palette. The goal is to make visible choices that fit this product.

## Start With A Reference

Ask for one real site, product, brand, screenshot, object, or place whose feel should guide the interface. If no reference is available, choose a named direction and state it.

Useful direction names:

- dense and operational
- editorial and text-forward
- stark technical tooling
- warm consumer utility
- playful interactive product
- premium product showcase
- calm administrative workspace
- high-contrast data room

Avoid direction names that mean nothing by themselves:

- modern
- clean
- sleek
- professional
- beautiful

## Choose Color

Pick color from a source:

- existing brand asset
- product photo or real-world object
- domain convention the audience already understands
- content state model such as risk, progress, severity, status, or category

Then define:

- page background
- primary action
- secondary action
- border and divider
- muted text
- emphasis or alert colors
- data or status colors if needed

Avoid using a framework swatch as the primary reason for a palette.

## Choose Type

Pick type for tone and readability:

- body face for reading
- display face only when the product needs a distinct editorial or brand voice
- monospaced face for code, logs, metrics, IDs, or technical density
- numeric behavior for tables, dashboards, prices, and counters

Do not leave type as a starter default unless the product intentionally wants invisibility over identity.

## Choose Layout

Answer these before drawing sections:

- What does the user need to do first?
- What must they trust before they act?
- What evidence should be visible immediately?
- What changes between empty, normal, error, and heavy-data states?

For tools and dashboards, prefer dense, scannable structure over marketing composition. For sites, make the first viewport specific to the product or subject, with a hint of the next section visible.

## Choose Details

Decide consciously:

- radius scale
- shadow depth
- border style
- icon style
- image treatment
- motion purpose
- empty-state tone

If the decision cannot be explained in relation to the product, user, or brand, simplify it.
